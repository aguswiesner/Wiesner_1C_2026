◆ app_main()

void app_main ( void )

Función principal de la aplicación.

Inicializa todos los periféricos necesarios (HC-SR04, LEDs, LCD, switches), configura las interrupciones para los switches, crea la tarea de medición y configura el timer para mediciones periódicas cada 1 segundo.

Esta función se ejecuta una vez al inicio del programa y luego termina, dejando que las tareas y interrupciones manejen la funcionalidad.
